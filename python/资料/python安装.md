#前置准备

##下载

先在[官网](https://www.python.org "python官网")下载最新的版本3.x版本

##安装

* 选择安装类型

![图片1](安装配图/1.jpg "选择安装类型")

如果选择默认安装请跳转到安装成功进行测试。**注意将py添加到环境变量**

* 自定义安装

![图片2](安装配图/2.jpg "选择自定义模块")

选择只定义安装的模块

* 选择安装路径

![图片3](安装配图/3.jpg "选择安装路径")

选择安装的路径 **install for all users 为安装的软件是否为全部用户使用**

* 安装成功

![图片4](安装配图/5.jpg "安装成功")

安装成功后显示这个界面点击Close关闭界面即可使用

##测试

![图片5](安装配图/6.jpg "测试")

安装完后进入命令行输入python -V

**注意V要是大写**

在命令行输入python就可进入python命令行模式

##配置环境变量

在“我的电脑-属性-高级系统设置-环境变量-系统变量”找到**Path**编辑python的完整路径
